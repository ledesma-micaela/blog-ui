# Blog Ui

# Pre requirements
- Have angular, node and npm installed
- Have this [API](https://github.com/ledesma-micaela/blog-backend) running

# Ejecution
- Clone repo
- Install dependencies: `npm install`
- Run aplication: `npm start`
- Open [localhost](http://localhost:4200)

# Routes
- `/posts-list`
- `/post/:id`
- `/admin/home`
- `/admin/add-post`
- `/admin/edit-post/:id`
